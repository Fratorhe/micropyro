============
Installation
============

Clone the repository from `Github <https://github.com/Fratorhe/micropyro>`_, in terminal::

    $ git clone https://github.com/Fratorhe/micropyro


Move in the cloned repository::

    $ cd micropyro

Install the package::

    $ pip install .

Or in development mode (recommended if modifications are going to be done)::

    $ pip install -e .
