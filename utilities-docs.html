utilities.reader_json_totals(list_filenames)[source]¶

This reads the json files with totals and returns them as a list of dicts. It will verify that the name of the file starts with totals.json to read it. This way, we can just send to the function all the files in the directory and it will take care of selecting the appropriate.

Returns
list_totals_dict: list dicts

list of dictionaries with the totals
